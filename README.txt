Woocommerce File Upload To Order from admin page

== Description ==

If you are looking a fast way for uploading files from admin order page then you are in the correct place.

some features :

upload any kind of file from admin order page
multiple upload
responsive support
preview uploaded images



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `es-woo-upload-files.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
